# Supermarket_Sales_Analysis
An end-to-end data analysis project focused on analyzing supermarket sales data to uncover business insights and improve decision-making. The project includes data cleaning, sales trend analysis, KPI tracking, and interactive dashboards built using Excel.
# Files Included
SuperMarrket_Dashboard.xlsx - Data set + Dashboard
# Dashboard Highlights
- Total Transaction
- Total Sales
- Profit
- Average Ratings
- Sales by product Category
- Payment Mode VS Sales
- Sales by Customer Type
# Tools Used
Microsoft Excel
# How to view this Dashboard
- Download and install Power BI Desktop
- Download the file SuperMarket_Dashboard.xlsx fromm the repo
- Open the file in Power BI Desktop
- Explore all visuals and interact with filters
